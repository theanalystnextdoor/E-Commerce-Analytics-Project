# 🛒 E-Commerce Analytics Dashboard (Power BI)

An end-to-end Power BI project transforming a flat transactional dataset into a structured, decision-ready analytics dashboard.

### 📊 Project Snapshot

* **3,500 transactions**
* **10 products | 4 regions**
* **Jan 2022 – Dec 2024**
* **$10.6M total revenue**
* **17.3% average net margin**

---

## 🏗️ What I Built

* Designed a **star schema data model** (Fact + 3 Dimensions)
* Created a fully custom **Date table (1,096 rows)** with fiscal logic
* Implemented **44 structured DAX measures**
* Built a **4-page executive dashboard**
* Applied a custom **dark JSON theme**

---

## 📐 Key Technical Skills Demonstrated

* Data Modeling (Star Schema)
* Advanced DAX (Time Intelligence, RANKX, CALCULATE, SUMX, ALL)
* KPI & Conditional Formatting Logic
* Target vs Actual Analysis
* Seasonal Trend Analysis
* Clean UI/UX Dashboard Design

---

## 🔍 Business Insights Delivered

* Identified highest-margin product despite low revenue rank
* Detected consistent seasonal performance patterns
* Compared regional targets vs actual performance
* Analyzed YoY and rolling growth trends
